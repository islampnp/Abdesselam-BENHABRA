## 📖 Content Guidelines

### Writing Style
- **Active voice:** "Delivered" not "responsible for"
- **STAR method:** Situation → Task → Action → Result
- **Quantified impact:** "8 engagements" vs "multiple projects"
- **No filler:** Remove generic phrases

### Case Study Format
```
Objective: Business problem & security challenge
My Role: Your specific responsibilities
Actions Taken: Tools, methodologies, process
Results/Impact: Quantified outcomes, client value
Tech Stack: Security tools used
```

### SEO Optimization
- Meta descriptions: Updated for each page
- Keywords: Naturally integrated (not stuffed)
- Headings: Proper H1 → H6 hierarchy
- Images: Descriptive alt text
- Internal links: Strategic cross-linking

## 🚨 Troubleshooting

### Portfolio Not Loading?
1. Clear browser cache (Ctrl+Shift+Delete)
2. Hard refresh (Ctrl+Shift+R)
3. Check internet connection
4. Verify URL spelling

### Images Missing?
1. Check file paths (relative vs absolute)
2. Verify case sensitivity (case-sensitive on servers)
3. Ensure files in assets/img/ folder
4. Check file format (.jpg not .JPG)

### Styling Issues?
1. Verify CSS files linked
2. Check file paths in <head>
3. Hard refresh browser
4. Check console for errors (F12)

**→ Full troubleshooting in [IMPLEMENTATION_GUIDE.md](IMPLEMENTATION_GUIDE.md)**
